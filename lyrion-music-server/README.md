## Setup Lyrion Music Server
I had two old Logitech Squeezebox radios. They were high quality and cutting edge at the time. 
Somewhere around 2018, Logitech decided to pull the plug on MySqueezebox.com which is required to run the radio. 
However they were kind enough to open-source the Logitech Media Server (LMS) code which was picked up by a small, yet dedicated community. 

https://lyrion.org/players-and-controllers/squeezebox-radio/

https://lyrion.org/

Once LXC is up and running, I SSH into it, downloaded the .deb and installed it using `apt install`.

The radios are working as new! 

Radio Paradise https://radioparadise.com/home is a great station for ad-free listening, playlists curated by real humans.
